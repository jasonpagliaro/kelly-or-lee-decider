# Kelly Or Lee Decider

A playful Next.js app that randomly chooses between two browser-local option names. It defaults to Kelly and Lee, stores custom names in `localStorage`, and keeps decision history only for the current page session.

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run test
npm run build
```

The app has no authentication, API routes, database, cookies, analytics, sound, or server-side persistence.
